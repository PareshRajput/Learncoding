CONTROL STATEMENTS:

In C++ programming, if statement is used to test the condition.\nThere are various types of if statements in C++. 1)if statement
2)if-else statement
3)Nested if statement
4)if-else-if ladder

If statement:


if(condition)
{
code to be executed
}

If-else statement


if( )
{
//codes to be executed if the condition is true
}
else( )
{
//codes to be executed if condition is false
}

Nested-if


if(condition 1)
{
if(condition 2)
{
//code will be executed if condition 2 is true
}
else( )
{
//if condition 2 is false
}
}
else( )
{
//codes will be executed if condition 1 is false
}

If-else-if ladder:


if(condition 1)
{
//code executed if condition 1 is true
}
else if(condition 2)
{
//code executed if condition 2 is true
}
else if(condition 3)
{
//code executed if condition 3 is true
}
else ( )
{
//code executed if none of
the three condition is true
}
