Text formatting:

We can use Bold, Italic and Underline tag anywhere, but we should close all tags properly.
For example if you have many nested tag you should close the tag in First Open Last Close format.
b .. /b - Bold your text
i .. /i - Italic text
u .. /u- Underline text
