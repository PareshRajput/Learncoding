•JavaScript is dynamic language.
•JavaScript provides different data types to hold different types of values.
•You need to use VAR here to specify the data type.
•It can hold any type of values such as numbers, strings etc.

1. Boolean - A Boolean represents only one of two values: true, or false. Example - var a = true; var b = false;
2. Number - There is only one type of Number in JavaScript. Numbers can be written with or without a decimal point. Example - var num1 = 32;
3. String - String data type represent textual data surrounding to single/double quotes. Strings are used for storing text. Example - var str1 = 'This is BCA First Year';
4. Null - Null has one value: null. It is explicitly nothing. Example - var nothing = null;
5. Undefined - JavaScript uninitialized variables value are undefined. Uninitialized variable (value undefined) equal to null. A variable that has no value is undefined. Example - var testVar; console.log(testVar);
